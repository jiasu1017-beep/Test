# PonyWork v0.0.5 发布说明

## 主要更新内容

### 新功能
- **RDP文件导入功能**：支持从本地RDP文件导入远程桌面连接配置，自动提取关键参数
- **远程桌面连接管理**：支持添加、编辑、删除、测试远程桌面连接
- **收藏功能**：支持收藏常用的远程桌面连接
- **应用集成**：支持将远程桌面连接添加到应用列表和集合

### 问题修复
- **RDP文件解析错误**：修复了配置值提取错误的问题
- **RDP文件导入错误**：修复了包含端口号的地址解析错误（如8.152.212.155:6019）
- **RDP配置项解析**：修复了主机地址和用户名等配置项导入时包含"s:"前缀的问题
- **编译环境配置**：修复了MinGW编译器路径问题

### 性能优化
- **连接测试**：优化了TCP连接测试的超时处理
- **文件解析**：优化了RDP文件解析算法，提高解析速度

## 安装说明
1. 下载发布包并解压
2. 运行PonyWork.exe启动应用程序
3. 首次运行时会自动创建必要的配置文件
4. 可通过"导入"按钮导入现有的RDP文件

## 系统要求
- Windows 7/8/10/11 (64位)
- .NET Framework 4.7.2或更高版本
- 至少2GB内存
- 至少100MB可用磁盘空间
- 网络连接（用于远程桌面连接）
