# PonyWork v1.0.1 发布说明

## 版本信息
- **版本号**: v1.0.1
- **发布日期**: 2026-03-18
- **发布类型**: 正式版

## 更新内容

### 新功能
- 添加日历视图功能并支持表格/日历视图切换
- 添加饼图双击查看详情功能
- 实现任务同步时处理删除任务的功能

### 问题修复
- 添加析构函数并修复动画指针空引用问题

### 样式调整
- 调整工作日志组件筛选控件的布局和宽度

### 代码重构
- 重构分类管理为下拉框并添加复选框委托
- 重构工作日志视图切换和日期选择功能
- 日历单元格悬停动画和任务详情弹窗

## 已知问题
- 部分 AI 模型需要代理才能访问（如 OpenAI、Claude、Gemini）
