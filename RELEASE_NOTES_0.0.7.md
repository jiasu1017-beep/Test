# PonyWork v0.0.7 版本更新说明

## 版本信息
- **版本号**: v0.0.7
- **发布日期**: 2026-03-02
- **更新类型**: 功能增强与问题修复

---

## 新功能实现

### 1. 设置界面全新改版
- 采用左侧导航栏 + 右侧内容区的布局设计
- 参考Trae设置菜单设计，提供更直观的设置体验
- 设置分类包括：通用设置、快捷键、AI设置、开机启动、检查更新、关于

### 2. AI设置深度整合
- **AI模型配置**：集成9种主流AI模型
  - MiniMax、OpenAI GPT-3.5/4、Claude-3、Google Gemini
  - 通义千问、讯飞星火、DeepSeek (硅基流动)、本地关键词匹配
- **API Key管理**：加密存储，保障用户隐私安全
- **API地址配置**：支持自定义API地址
- **连接测试**：一键测试AI配置是否正确
- **AI对话测试**：内置AI对话测试窗口
- **功能开关**：可控制智能任务分析、AI报告生成、智能建议等功能

### 3. Word导出优化
- 采用Word COM自动化技术，实现真正的DOCX文件生成
- 解决之前导出文件过小、无法正常打开的问题
- 支持更丰富的文档格式设置

### 4. 更新检查优化
- 检查更新后不再弹出"已是最新版本"提示框
- 改为在状态栏显示，3秒后自动消失

---

## 问题修复

1. **段错误(SIGSEGV)修复**
   - 修复设置界面创建时的空指针问题
   - 确保所有UI控件正确设置父对象

2. **Word导出文件损坏**
   - 之前导出的Word文件无法正常打开
   - 原因：HTML格式转换问题
   - 解决方案：改用Word COM自动化技术

3. **AI对话显示优化**
   - 将AI名称从固定的"AI"改为当前使用的大模型名称
   - 对话中显示"🤖 通义千问:"、"🤖 DeepSeek:"等具体模型名称

## 升级注意事项

1. **数据兼容性**: v0.0.7 版本完全兼容 v0.0.6 版本的数据文件
2. **AI配置迁移**: 如有AI配置，需要重新输入API Key并保存
3. **首次使用AI**: 请在"设置 → AI设置"中配置您的AI模型和API Key
