# PonyWork 小马办公 - 功能介绍

## 软件简介

**小马办公 (PonyWork)** 是一个功能完善的桌面办公助手应用，帮助您更高效地管理和使用电脑。基于 Qt 框架开发，支持 Windows 7/8/10/11 (64位)。

## 📱 应用管理

### 主要功能
- **添加应用**：浏览并选择 .exe 文件，添加应用程序
- **编辑应用**：修改应用名称、图标、路径等信息
- **删除应用**：删除不需要的应用条目
- **启动应用**：双击或点击按钮启动应用

### 增强功能
- **视图模式**：支持大图标和列表两种视图模式
- **多选操作**：
  - Shift + 点击：连续选择
  - Ctrl + 点击：逐个选择
  - 鼠标拖拽：框选
- **排序功能**：按名称、添加时间排序
- **收藏功能**：收藏常用应用，方便快速访问
- **批量操作**：批量启动和批量删除应用
- **图标获取**：自动从可执行文件提取应用图标

### AI 图标生成
- 集成多种AI服务（Flux、OpenAI、通义千问、DeepSeek等）
- 支持9种主流AI模型生成个性化图标
- 一键生成精美应用图标
- AI日志记录，方便问题排查

---

## 📂 集合管理

### 主要功能
- **创建集合**：自定义应用分组，命名集合
- **编辑集合**：修改集合名称和包含的应用
- **删除集合**：删除不需要的集合
- **批量启动**：一键启动集合中的所有应用
- **集合排序**：调整集合显示顺序
- **自动选择**：启动时自动选择第一个集合

## 📦 安装说明

### 安装步骤
1. 下载发布包并解压
2. 运行 `PonyWork.exe` 启动应用程序
3. 首次运行时会自动创建必要的配置文件

### 首次配置建议
1. **AI 功能**：在设置中配置 AI API Key 以启用 AI 图标生成功能
2. **远程桌面**：确保系统远程桌面功能已启用
3. **应用导入**：导入常用的应用程序到应用管理器
