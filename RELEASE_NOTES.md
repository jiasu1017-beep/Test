# PonyWork v0.0.5 版本更新说明

## 新功能实现

1. **RDP文件导入功能优化**
   - 支持从本地RDP文件导入远程桌面连接配置
   - 以文件名作为远程桌面连接名称
   - 分类默认为"工作"
   - 自动提取服务器地址、端口、用户名等关键连接参数

2. **远程桌面连接管理**
   - 支持添加、编辑、删除远程桌面连接
   - 支持连接测试功能
   - 支持收藏常用连接
   - 支持将连接添加到应用列表和集合

## 问题修复

1. **RDP文件解析错误**
   - 修复了使用固定长度截取配置值导致的解析错误
   - 改用动态查找冒号位置的方法，确保正确提取配置值
   - 支持处理各种格式的RDP配置项

2. **RDP文件导入错误**
   - 修复了包含端口号的地址解析错误（如8.152.212.155:6019）
   - 优化了配置项解析逻辑，使用双冒号定位方法
   - 确保正确提取服务器地址、端口、用户名等关键参数

3. **RDP配置项解析**
   - 修复了主机地址和用户名等配置项导入时包含"s:"前缀的问题
   - 改进了配置值提取算法，跳过配置类型前缀（如s:、i:等）
   - 提升了RDP文件导入的准确性和可靠性

4. **编译环境配置**
   - 修复了MinGW编译器路径问题
   - 优化了编译脚本，确保构建过程稳定可靠

## 性能优化

1. **连接测试性能**
   - 优化了TCP连接测试的超时处理
   - 添加了进度对话框，提升用户体验

2. **文件解析性能**
   - 优化了RDP文件解析算法，提高解析速度
   - 减少了内存使用，提升了大文件处理能力

## 已知问题

1. **音频重定向**
   - 部分系统可能存在音频重定向不稳定的问题

2. **打印机重定向**
   - 某些打印机型号可能无法正确重定向

3. **性能优化**
   - 在高分辨率显示器上，远程桌面连接可能会出现轻微延迟

## 安装说明

1. 下载发布包并解压
2. 运行PonyWork.exe启动应用程序
3. 首次运行时会自动创建必要的配置文件
4. 可通过"导入"按钮导入现有的RDP文件

## 系统要求

- Windows 7/8/10/11 (64位)
- .NET Framework 4.7.2或更高版本
- 至少2GB内存
- 至少100MB可用磁盘空间
- 网络连接（用于远程桌面连接）
