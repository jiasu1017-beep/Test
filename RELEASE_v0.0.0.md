## 🚀 本次发布内容

### 新功能

1. **"关于小马办公"对话框高度优化**
   - 添加滚动区域支持，确保内容在小屏幕上也能完整显示
   - 设置合适的最小和最大尺寸（480x600最小，900高度上限）
   - 优化布局结构，提升界面美观度
   - 调整图片尺寸，更适合不同屏幕分辨率

2. **"应用集合"标签页自动选择功能**
   - 打开"应用集合"标签页时，自动默认选择列表中的第一个应用集合
   - 直观显示选择状态
   - 自动加载并显示第一个应用集合的相应内容
   - 每次打开标签页都保持一致的默认选择行为

### 改进点

- 增强了界面在不同屏幕尺寸下的适配性
- 提升了用户体验，减少了手动选择操作
- 优化了代码结构，确保功能稳定性

### 已知限制

- 暂无已知的严重Bug
- 部分功能仍在持续优化中
