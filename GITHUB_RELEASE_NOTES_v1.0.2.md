## 更新内容

- feat(任务管理): 重构任务分类为内置类型并支持多用户 (fd14904)
- refactor(worklogwidget): 移除任务操作前的登录状态检查 (c143a96)
- fix(任务同步): 修复用户切换时任务数据同步问题 (d8a0997)
- feat(用户模块): 添加HTTP请求日志记录功能 (ca6d736)
- refactor(admin-panel): 重构日志管理页面使用 Bootstrap Tab 组件 (038be48)
- feat(app管理): 添加应用排序功能并优化UI (437ade7)
- refactor(widgets): 修改排序模式文本和按钮样式 (3100e4e)
